## Преимущества новой структуры

1. **DRY принцип** - нет дублирования данных и кода
2. **Единый источник истины** - все данные в одном месте
3. **Легкость поддержки** - изменения в одном месте применяются везде
4. **Типобезопасность** - улучшенная типизация TypeScript
5. **Масштабируемость** - легко добавлять новые данные и константы
6. **Читаемость** - более понятная структура проекта

## Структура проекта

```
src/
├── constants/          # Константы и конфигурация
│   ├── navigation.ts  # Навигационные ссылки
│   └── company.ts     # Информация о компании
├── config/            # Конфигурация приложения
│   └── routes.ts      # Маршруты
├── types/             # TypeScript типы
│   └── index.ts       # Базовые типы
├── data/              # Mock данные
│   └── mockData.ts    # Тестовые данные
├── components/        # React компоненты
├── pages/             # Страницы
└── hooks/             # Custom хуки
```
