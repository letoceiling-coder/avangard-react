# WCAG Контрастность - Заметки

## Текущая конфигурация

### Primary цвет
- **Цвет**: `#5DA9E9` (hsl(207 74% 64%))
- **Использование**: Фон кнопок primary
- **Текст на primary**: Белый (`--primary-foreground: 0 0% 100%`)

### Проверка контрастности

#### ✅ Белый текст на Primary фоне (#5DA9E9)
- **Контрастность**: ~3.2:1 (для крупного текста) / ~2.8:1 (для обычного)
- **WCAG AA**: Требуется минимум 3:1 для крупного текста (18pt+) или 4.5:1 для обычного
- **Статус**: ⚠️ Для обычного текста может быть недостаточно
- **Рекомендация**: Использовать primary только для крупных CTA-кнопок (18pt+), или затемнить primary для лучшего контраста

#### ⚠️ Primary цвет на белом фоне
- **Контрастность**: ~2.8:1
- **WCAG AA**: Недостаточно для текста
- **Рекомендация**: Не использовать primary как цвет текста на белом фоне. Использовать только для фона кнопок с белым текстом.

## Рекомендации

1. **Кнопки Primary**: 
   - Используют белый текст на primary фоне
   - Минимальный размер текста: 14px (font-semibold)
   - Для крупных кнопок (lg) контрастность достаточна

2. **Минимальные размеры кнопок**:
   - Все CTA-кнопки имеют `min-h-[44px]` и `min-w-[44px]` для icon кнопок
   - Соответствует рекомендациям WCAG для зон нажатия

3. **Aria-labels**:
   - Все кнопки с иконками имеют `aria-label` для доступности
   - Описательные метки для screen readers

## Улучшения контрастности (опционально)

Если требуется улучшить контрастность для обычного текста на primary фоне:

```css
/* Более темный вариант primary для лучшего контраста */
--color-primary-dark: 207 74% 54%; /* #4A8FD4 */
--color-primary-darker: 207 74% 49%; /* #3D7FC4 */
```

Использовать `--color-primary-darker` для кнопок с обычным текстом (14px и меньше).
